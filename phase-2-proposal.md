## 2. Phase 2 design

### Framing

Phase 2 goal: build from run-walk to **continuous 5k**, then **continuous 10k**. These are the Phase 2→3 gate criteria. Don't shortcut them by switching to continuous running before the intervals earn it.

Parallel tracks running throughout:
- **Strength:** shift from pure rehab to functional load. Single-leg work, RDL, lateral band walk evolve toward heavier, more athletic versions. Core progresses as diastasis permits.
- **Ring pull-up:** continues as a dedicated apparatus block in every session. Not phase-dependent.
- **Gymnastics beyond rings:** begins to weave in as knee stability and glute strength demonstrate robustness. No fixed timeline.

Estimated duration of Phase 2: **12–20 weeks**, paced by the knee and diastasis response to impact load — not the calendar.

---

### 2a. Run-walk progression toward continuous 5k

No treadmill means running on sprung floor (ideal — low impact, consistent surface) or roads. Early sessions on the sprung floor are preferred while the knee is adapting to impact.

The progression follows four blocks. Advance a block only when the previous block's sessions consistently produce knee ≤ 2/10 during and ≤ 2/10 at 24h, and no diastasis symptoms under impact.

#### Block A — Establish the habit (target: ~3 weeks)

Format: **1 min run / 2 min walk**

| Session | Run-walk rounds | Total active time |
|---|---|---|
| A1 | 5 × (1 min run / 2 min walk) | ~15 min |
| A2 | 5 × (1 min run / 2 min walk) | ~15 min — repeat A1 if knee above 2/10 |
| A3 | 6 × (1 min run / 2 min walk) | ~18 min |
| A4 | 6 × (1 min run / 2 min walk) | ~18 min — consolidate before advancing |

Gate to Block B: two consecutive Block A sessions with knee ≤ 2/10 during and at 24h.

#### Block B — Extend run intervals (target: ~3–4 weeks)

Format: **2 min run / 2 min walk**, then **2 min run / 90 sec walk**

| Session | Format | Total active time |
|---|---|---|
| B1 | 5 × (2 min run / 2 min walk) | ~20 min |
| B2 | 5 × (2 min run / 2 min walk) | ~20 min |
| B3 | 5 × (2 min run / 90 sec walk) | ~17 min |
| B4 | 6 × (2 min run / 90 sec walk) | ~21 min |

Gate to Block C: two consecutive Block B sessions with knee ≤ 2/10 at 24h, and diastasis controllable under impact.

#### Block C — Build run segments (target: ~3–4 weeks)

Format: increasing run intervals, decreasing walk intervals

| Session | Format | Total active time |
|---|---|---|
| C1 | 5 × (3 min run / 2 min walk) | ~25 min |
| C2 | 4 × (5 min run / 2 min walk) | ~28 min |
| C3 | 3 × (8 min run / 2 min walk) | ~30 min |
| C4 | 2 × (10 min run / 2 min walk) | ~24 min — assess how continuous 10 min feels |

Gate to Block D: two consecutive Block C sessions comfortable, knee ≤ 2/10 throughout.

#### Block D — Approaching continuous 5k (target: ~3–4 weeks)

| Session | Format | Notes |
|---|---|---|
| D1 | 20 min continuous easy run | First continuous 20 min. Pace is irrelevant — completion matters. |
| D2 | 25 min continuous | If D1 knee ≤ 2/10 at 24h. |
| D3 | 30 min continuous | |
| D4 | 5k attempt | Time doesn't matter. No walk breaks. Earn the waypoint. |

**Pace guidance for all blocks:** conversational — able to speak in full sentences. If asthma is triggered, slow to a walk immediately, resume when comfortable, log the conditions. Do not push through breathing difficulty.

### 2b. Session structure with 2–3 sessions per week

The available days are Monday, Thursday, Saturday. The split below is designed for 3 sessions, with a 2-session fallback.

#### 3 sessions per week (optimal)

| Day | Session type | Primary content | Duration |
|---|---|---|---|
| Monday | Strength-primary | Full glute/posterior chain block; dead bug core progression; ring apparatus | 60–75 min |
| Thursday | Run-walk primary | 5 min walk warm-up; run-walk block (Block A–D progression); maintenance strength (step-downs, RDL); ring apparatus | 60–75 min |
| Saturday | Combined | Moderate strength (not max volume); medium run-walk; ring apparatus | 75–90 min |

Monday is deliberately not a run day: it lets the weekend run-walk dissipate before loading legs again. Thursday's run lands mid-week so there are two full recovery days before Saturday.

#### 2 sessions per week (fallback)

| Session | Primary content |
|---|---|
| A | Full strength block + short run-walk (Block A or B only, max 20 min) + rings |
| B | Run-walk primary (full progression block) + minimal strength (glute bridge, step-down) + rings |

When only two sessions are available in a week, do not skip the run-walk session in favour of strength — the run-walk progression has more time-sensitivity given the marathon timeline.

### 2c. Ring pull-up parallel track — integration without shoulder overload

Rings are upper body / scapular work. Running is lower body / cardiovascular. They don't directly conflict. The ring apparatus block is a constant 8–10 min fixture in every session regardless of session type.

**Placement within sessions:**
- Run-walk primary sessions: rings go **after** running. Fresh legs for the run; upper-body fatigue from rings doesn't affect running mechanics.
- Strength-primary sessions: rings go at the **end** — after the posterior chain and knee work, so the scapular depression required in the ring support hold isn't fatigued by earlier pulling movements.

**Phase 2 ring progression targets** (building on Phase 1):

| Stage | Phase 1 exit target | Phase 2 target |
|---|---|---|
| Passive dead hang | 3 × 30 sec | 3 × 45 sec → 3 × 60 sec |
| Active scapular pull | Beginning — initiate pull from depressed scaps | 3 × 5 controlled scapular pull-downs (no elbow bend) |
| Foot-assisted ring support hold | 3 × 15 sec (feet on box) | Progressing to 3 × 15 sec with feet only lightly touching (majority load through arms) |
| Band-assisted pull-up | Not started | Introduce in Phase 2 mid-point: heaviest band, full ROM, 3 × 3–5 |

Ring progression rule: advance a stage when the current stage is comfortable **across two consecutive sessions** — not on the calendar.

At 105kg bodyweight, the pull-up track will likely extend into Phase 3. This is expected. Do not rush scapular control to chase the band-assisted work.

## 4. What to decide before integrating this proposal

1. **GP conversation.** A brief check-in about starting a running programme is the lowest-friction way to close the cardiovascular sign-off question. This doesn't require a referral — just a call or message.

2. **Confirm the Phase 1 gate result after Session 5.** The session 5 log needs to show: knee ≤ 2/10 on step-down and squat; pelvic control on single-leg bridge improving; diastasis still manageable. Ben fills in the gate table — review it before proceeding.

3. **Confirm the run surface.** Sprung floor is the preferred early run surface. If the gymnastics club floor is available on all three session days, all early run-walk work can stay there before transitioning outdoors.

4. **Progression cadence.** The run-walk blocks above assume roughly one advancement per 2 sessions. This is conservative. If knee stays well below 2/10 consistently, progressions could be weekly. If knee creeps toward 2/10, slow down — do not advance blocks until two consecutive clean sessions.

5. **Diastasis under impact.** Running is a new load on the abdominal wall. The first few run-walk sessions should include a self-check: any sensation of downward pressure, bulging, or discomfort along the linea alba during or after the run intervals. If yes, note it, reduce the run interval duration by half, and build back up.
